# 更新日志

## [1.2.0] - 2025-11-11

### 新增功能

- 完整的 macOS 原生应用支持
- 拖拽文件处理功能
- 在线视频 URL 处理 (YouTube, Bilibili 等)
- 语音转写功能 (集成 whisper.cpp)
- 说话人识别功能
- 多语言翻译功能 (NLLB-200 模型)
- 字幕导出功能 (SRT, VTT, ASS 格式)
- 国际化支持 (zh-TW, zh-CN, en, ja, ko)
- 深色模式支持
- 系统通知集成
- 全局热键支持
- 模型按需下载管理
- 剪贴板监控功能
- 字幕样式自定义
- 字幕编辑功能
- 字幕分享功能

### 技术特性

- Flutter Desktop 框架
- FFI 集成本地库
- App Store 合规支持
- 沙盒环境适配
- 原生菜单栏集成
- Touch Bar 支持

### 性能优化

- 内存占用优化 (< 2GB)
- 启动时间优化 (< 2 秒)
- 转写性能优化 (1 分钟音频 ≤ 10 秒)
